# iRite for Visual Studio Code

This extension adds support for the iRite Language to Visual Studio Code. It supports:

* Colorization
* Snippets
* Preprocessing
* External compiler

# Installation

Press `F1` in VSCode, type `ext install` and then look for `Language iRite`.
